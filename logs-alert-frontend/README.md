# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.
- Migrated from CRA

## Client Start 
1. Clone the full repo

2. `npm install` inside the frontend

3. `npm run dev`

## ENV
1. Create a .env file inside the root direc and set VITE_API_URL
2. `VIE_API_URL = "127.0.0.1:8000"`
